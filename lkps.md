# Lembar Kerja Program Studi (LKPS)

LKPS adalah instrumen resmi yang dikeluarkan oleh **LAM Teknik** untuk mengumpulkan data kuantitatif dan kualitatif dari Program Studi dalam rangka asesmen akreditasi.

## Fungsi LKPS
- Menjadi dasar verifikasi oleh asesor selama visitasi
- Menyediakan data objektif untuk evaluasi capaian standar
- Memastikan keseragaman format pelaporan antar institusi

## Komponen Utama LKPS
| Bagian | Isi Utama |
|-------|----------|
| **A. Profil** | Nama PS, jenjang, akreditasi sebelumnya, jumlah mahasiswa/dosen |
| **B. Mahasiswa & Lulusan** | Daya tampung, IPK, masa studi, tracer study, employability |
| **C. Sumber Daya Manusia** | Kualifikasi dosen, rasio dosen:mahasiswa, beban mengajar |
| **D. Kurikulum & Pembelajaran** | Struktur kurikulum, CPL, metode evaluasi |
| **E. Sarana & Prasarana** | Laboratorium, perpustakaan, akses IT |
| **F. Penelitian & Pengabdian** | Jumlah judul, publikasi, HKI, dana penelitian |
| **G. Kerja Sama** | Daftar MoU, jenis kegiatan, luaran |

## Pengisian dan Verifikasi
- Data diisi berdasarkan **sistem informasi akademik resmi (SIAKAD)**
- Semua angka diverifikasi melalui **dokumen pendukung** (SK, laporan, sertifikat)
- Diunggah ke **SISTER LAM Teknik** sesuai jadwal
